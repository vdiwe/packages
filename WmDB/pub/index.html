webMethods IS Database Manager
 

Documentation

This is the Home Page for the WmDB package.  Via a startup service, the package registers itself with the main page of the webMethods Integration Server.  There will be a Database button on the left hand side of the page that brings up the database service generation pages.  The WmDB package contains the pub.db services that used to be part of the WmPublic package.

Each of the tabs of the Database administration page contains links to help pages.

server.cnf properties:

watt.server.db.provider
watt.server.jdbc.defaultDriver
watt.server.jdbc.driverList
watt.server.jdbc.moreResults
watt.server.jdbcLogging
watt.server.jdbcLogFile
