Screen Descriptions : Alias Management Screen
Alias Management Screen
Screen Heading: WmDB > Alias Management
Use the Alias Management screen to add, edit, and delete database aliases. A database alias specifies information that the Integration Server needs to connect to a database.
To access this screen, click Alias Management in the WmDB Navigation panel.
Current Data Sources
Lists database aliases that have been defined to the server.
Add
Opens the New DB Alias Screen, allowing you to add a new database alias.
Edit
Opens the Edit Alias Information Screen, allowing you to update connection information for a database alias.
Delete
Deletes the selected database alias.
Connect
Tests the database connection.
