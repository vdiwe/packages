Screen Descriptions : Tables in (DataSourceName) Screen
Tables in (DataSourceName) Screen
Screen Heading: WmDB > Service Generation > Connection Parameters > Tables
Use the Tables in (DataSourceName) screen to create a new service by selecting the table that you want the service to access and the database operation that you want the service to perform.
To access this screen, open the Specify Connection Parameters Screen. Use the Restrictions fields to limit the names and types of tables with which you want to work, if necessary, and click Connect. The Tables in (DataSourceName) screen is displayed, listing the tables available in the database.
Tables in Datasource
Click on a table name to select the table you want the service to access. The server displays the name and type of the tables in the selected database. The Catalog and Schema fields contain information about the tables only if you defined these fields on the Specify Connection Parameters Screen.
Column Name
Description
Name
Displays the names of the tables that are in the database. Click the name of the table that you want the service to access; the server displays the Columns in (TableName) Screen.
Type
Displays the type of each table in the database, e.g., Table, View, System Table.
Catalog
Displays the classification of each table in the database.
Schema
Displays the table description used to subset and organize each table in the database.
Remarks
Displays an explanatory comment about the table.
Test SQL Queries
Use the field in this section to submit a SQL statement to the database. Type the SQL statement in the box, then click Execute query.
